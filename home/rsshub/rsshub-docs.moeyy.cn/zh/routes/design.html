Axis Studios​

Work type​

作者：@MisteryMonster

举例：https://rsshub.moeyy.cn/axis-studios/work/full-service-cg-production

路由：/axis-studios/:type/:tag?

参数：

  • type，必选 — work, blog
  • tag，可选 — Work type URL: compositing, full-service-cg-production, vfx-supervision, realtime, art-direction, animation

Work type URL in articles. Such as： 'https://axisstudiosgroup.com/work/full-service-cg-production' the tag will be full-service-cg-production.

Some tags are rarely used： Script, direction, production, design-concept etc。

发现​

支持浏览器扩展支持 RSSBud

作者：@nczitzk

举例：https://rsshub.moeyy.cn/zcool/discover

路由：/zcool/discover/:query?/:subCate?/:hasVideo?/:city?/:college?/:recommendLevel?/:sort?

参数：

  • query，可选 — 查询参数或分类，若填写分类见下表，默认为空 或 0 即精选
  • subCate，可选 — 子分类，见下表，默认为 0 即该父分类下全部
  • hasVideo，可选 — 是否含视频，默认为 0 即全部，亦可选 1 即含视频
  • city，可选 — 地区代码，填入发现页中 选择城市 中的各级地名，如 亚洲、中国、北京、纽约、巴黎等
  • college，可选 — 学校，默认为 0 即全部
  • recommendLevel，可选 — 推荐等级，见下表，默认为 2 即编辑精选
  • sort，可选 — 排序方式，可选 0 即最新发布 或 9 即默认排序，默认为 9

查看 精选 分类下的全部内容，其他参数选择默认，可直接使用路由 /zcool/discover/0

查看 精选 分类下的 运营设计 子分类全部内容，其他参数选择默认，可直接使用路由 /zcool/discover/0/617

在 精选 分类下的 运营设计 子分类全部内容基础上，筛选出有 视频，可直接使用路由 /zcool/discover/0/617/1

在 精选 分类下的 运营设计 子分类全部内容基础上，筛选出有 视频，且城市选择 北京，可直接使用路由 /zcool/discover/0/617/1/北京
